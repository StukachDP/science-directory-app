В текущей папке представлены файлы, 
которые задают стилевой оформление компонентам в react приложении.
Фронт-енд данного приложения использует фреймворк bootstrap-react.
Теги, структурирующие блоки, контенеры и поля форм и так далее импортированы с bootstrap.
Во всех файлах стилевого оформления используется react модуль "styled-components",
Который позволяет писать собственные css стили к различным тегам при этом соблюдая структуру и методологию react приложения.
Если в некоторых блоках и контейнерах в определенных файлах нет css стилей, это значит, 
что применяются собственные стили bootstrap.
Ссылка на документацию по модулю "styled-components": https://github.com/styled-components/styled-components
Ссылка на сайт фреймворка react-bootstrap: https://react-bootstrap.github.io/components
